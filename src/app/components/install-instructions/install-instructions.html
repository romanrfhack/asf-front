Instalar App ASFM

Para Android:

  1. Abre Chrome y navega a nuestro sitio
  2. Toca el menú (tres puntos en la esquina superior derecha)
  3. Selecciona "Agregar a pantalla de inicio" o "Install app"
  4. Confirma la instalación

Para iPhone/iPad:

  1. Abre Safari y navega a nuestro sitio
  2. Toca el botón de compartir (cuadrado con flecha hacia arriba)
  3. Desplázate hacia abajo y selecciona "Agregar a pantalla de inicio"
  4. Confirma la instalación

Beneficios de la App:

  • Acceso rápido desde tu pantalla de inicio
  • Funciona sin conexión a internet
  • Experiencia similar a una app nativa
  • Actualizaciones automáticas
  • Notificaciones push (próximamente)
